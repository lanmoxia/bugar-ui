# 开始使用

请先 [安装](#/doc/install) 本组件库。

然后在您的代码中写入下面的代码，就可以使用我提供的组件了。

```
import {Button, Tabs, Tab, Switch, Dialog, openDialog} from "bugar-ui"
```

## Vue 单文件组件

代码示例：

```
<template>
 <div>
  <Button>按钮</Button>
 </div>
</template>

<script setup lang="ts">
  import {Button, Tabs, Tab, Switch, Dialog, openDialog} from "bugar-ui"
</script>
```
